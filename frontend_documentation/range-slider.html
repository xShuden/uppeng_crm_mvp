RangeSlider

  1. Forms
  2. RangeSlider
OverviewOfficial Website

A fast & lightweight react component as a drop in replacement for HTML5 input range slider element.

CSS
<!-- RangeSlider css -->
import "react-rangeslider/lib/index.css";
Import Package
<!-- RangeSlider-->
import Slider from "react-rangeslider";
Add Package
yarn add react-rangeslider --save
Remove Package
yarn remove react-rangeslider or you can remove package by removing specific package from package.json
